Important FAQ

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for HTML documents.It treats the HTML document as a tree structure where each part of the document(such as heading, paragraph, images etc.) is represented as a node.

2. How do you select an element from the DOM?

Select the elements from the DOM using Javascript in the most common 4 ways:
1.getElementById()
2.getElementsByClassName()
3.getElementsByTagName()
4.querySelector() & querySelectorAll()

3. What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is basically a pattern to handler events efficiently.When an event occurs on a child element, it first triggers the event listener on that element.Instead of adding an event listener to each and every similar elements.It simplifies your code by centralizing event handling, reducing repetition,and avoiding issues like adding unneccessary listeners.
